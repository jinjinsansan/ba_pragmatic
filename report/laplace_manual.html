Sync Mode の使い方と動作の見方

🚀 起動手順

1 GUI を起動

GUIアイコンをダブルクリック。Electronウィンドウが起動します。

2 Settings を開く

右上の歯車アイコンをクリック。

3 BOT Tab で設定
Chip Base1
Profit Target30
Loss Cut3000
BET ModeSync (Top tables + regularity monitor)
Dry Runオフ（本番）

SAVE をクリック。

4 START ボタンをクリック

resume か 新規かのダイアログが出たら、状況に応じて選択。初回は「新規」。

⚠️ トラブルシューティング

Q1. ロビーで長時間待機している

A: 4テーブル全てが35ハンド未満の場合、入場条件を満たすまで待機します（正常動作）。通常10-20分程度。

Q2. SESSION EXPIRED が出た

A: 60秒ごとのハートビートで自動検出・自動復旧します。放置OK。3分以上経過してもBETしない場合は手動でSTOP→STARTを推奨。

Q3. ブラウザが固まった

A: full_recovery機能が発動し、page.gotoで完全リロードします。それでもダメならGUIを再起動してください。

Q4. テーブル変更が頻繁

A: シュー切替を即検出して退避しています（正常動作）。1-2分おきに変更される場合があります。

🔧 自動進化システム

毎日 AM 6:00 にVPSで自動分析が走ります：

  1. VPS の analytics.sqlite3 を分析
  2. 過去全データで100%勝率テーブルを抽出
  3. Supabaseに保存
  4. GUI起動時に最新リスト取得
✅ Evolutionが対策してきても翌日には新しいTop Nテーブルに自動切替されます。ユーザーは何もする必要なし。
