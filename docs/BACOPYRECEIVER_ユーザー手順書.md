## 1. 動作環境

### 必須要件

| 項目 | 要件 |
|---|---|
| OS | Windows 10 / 11（64bit）|
| メモリ | 8GB 以上推奨 |
| ストレージ空き | **2GB 以上**（インストール時1.4GB 使用） |
| ネットワーク | 常時接続（光回線推奨、モバイル回線不可） |
| ディスプレイ | 1920×1080 以上推奨 |
| 権限 | 一般ユーザー権限で可（管理者権限不要）|

### 非推奨環境

- VPN・プロキシ経由でのインターネット接続
- ウイルス対策ソフトの「強力な保護」モード（後述の例外設定が必要）
- 複数のディスプレイで仮想デスクトップを頻繁に切替える環境

## 2. インストール手順

### Step 1: インストーラをダウンロード

運営から配布されたリンク（Google Drive 等）から  
**`BACOPYRECEIVER Setup 0.1.0.exe`** (約 580 MB) をダウンロードしてください。

ダウンロード時間の目安:
- 光回線: 1〜3 分
- 集合住宅: 5〜10 分

### Step 2: SmartScreen の警告を解除

ダウンロードした EXE をダブルクリックすると、以下のような青い画面が出ることがあります:

```
┌─────────────────────────────────────────┐
│ WindowsによってPCが保護されました       │
│                                         │
│ Microsoft Defender SmartScreen は、認識  │
│ されていないアプリの起動を停止しました。│
│                                         │
│  [詳細情報]                             │
└─────────────────────────────────────────┘
```

1. 青い枠内の **「詳細情報」** をクリック
2. 出現する **「実行」** ボタンをクリック

> **なぜ警告が出るの？**: コード署名証明書を付けていないため Windows が「見知らぬアプリ」と判断します。無害です。

### Step 3: インストール実行

1. インストーラが起動し「BACOPYRECEIVER をインストールしています...」と表示されます
2. **約 1〜2 分** でインストール完了（956MB の Camoufox Firefox を展開するため）
3. 完了すると自動的にアプリが起動し、デスクトップに **`BACOPYRECEIVER`** のアイコンが作成されます

### Step 4: インストール先の確認

デフォルトで以下にインストールされます:

```
C:\Users\<あなたのユーザー名>\AppData\Local\Programs\bacopy-copytrade-gui\
```

エクスプローラで見たい場合は、アドレスバーに以下を貼り付け:

```
%LOCALAPPDATA%\Programs\bacopy-copytrade-gui
```

## 3. 初回起動と初期設定

### Step 1: アプリを起動

デスクトップの **`BACOPYRECEIVER`** アイコンをダブルクリック。

初回起動時は **Firefox データの展開**に 30 秒〜1 分かかります（2回目以降はすぐ起動）。

### Step 2: ログイン

起動すると **SIGN IN** 画面が表示されます:

| 入力欄 | 内容 |
|---|---|
| メールアドレス | 運営から発行された bafather.uk アカウントのメール |
| パスワード | 同上 |

「サインイン」ボタンをクリック → ダッシュボード画面に遷移します。

### Step 3: Stake アカウント情報を入力

ダッシュボードで Stake アカウントの情報を入力します:

| 項目 | 内容 |
|---|---|
| Stake ユーザー名 | Stake のログイン ID |
| Stake パスワード | Stake のログインパスワード |
| 実行者ラベル | 任意の名前（例: 自宅PC）|
| BET モード | 運営の指示に従って選択 |

> **セキュリティ注意**: パスワードは暗号化して保存されます。Stake のパスワードは、他サイトで使い回しているものは**絶対に避けて**ください。BACOPYRECEIVER 専用のパスワードを新しく設定することを強く推奨します。

### Step 4: BET 開始

**「START BOT」** ボタンを押すと:

1. バックグラウンドで Camoufox（カスタマイズされた Firefox）が起動
2. Stake にログイン（初回は 2 分ほどかかることあり）
3. Pragmatic Play のロビーを開く
4. 指示されたテーブルに着席し、BET を自動実行

> **初回ログイン時の確認メール**  
> Stake から「新しいログインを検出しました」というメールが届く場合があります。これは正常です。ブラウザのフィンガープリントが自動生成されているためで、メール内の「あなたでしたか？」確認は **「はい」** と回答してください。

## 4. 日常の操作

### 毎日の起動・停止

- **起動**: デスクトップアイコンをダブルクリック → アプリが起動 → 「START BOT」
- **停止**: アプリ内の「STOP BOT」ボタン → 右上 × でアプリ終了

### 24時間稼働させたい場合

PCをスリープ・シャットダウンしない状態で放置してください。アプリは自動的に:

- 切断時に再接続
- 定期的なページ更新（6時間ごと）
- アイドル時の人間らしい操作エミュレーション

を行います。

### 稼働状況の確認

アプリ内の **SIGNAL パネル** で以下が見えます:

| 表示 | 意味 |
|---|---|
| CYCLE | 現在の BET サイクル |
| RATIO | 勝敗レシオ |
| DRIFT | 予測からのズレ |
| HEARTBEAT | ボットが生きているか（緑で点滅）|

### 残高確認

ダッシュボードの **「BALANCE」** セクションで Stake の残高と BACOPY 側の料金残高が確認できます。

## 5. よくあるトラブルと対処

### ❶ アプリが起動しない

**症状**: デスクトップアイコンをダブルクリックしても何も起きない

**対処**:
1. 既に起動中の可能性 → タスクマネージャで `BACOPYRECEIVER` を探す、あれば一度終了
2. ウイルス対策ソフトがブロックしている可能性 → 例外追加:
   - Windows Defender: 「ウイルスと脅威の防止」→「設定の管理」→「除外の追加」→以下フォルダを追加:
     ```
     C:\Users\<あなた>\AppData\Local\Programs\bacopy-copytrade-gui
     ```

### ❷ 「Permission denied (publickey)」ログが出る

**症状**: ログに Support tunnel 関連のエラー

**対処**: 遠隔サポート機能の通信エラーですが、BET 動作には影響ありません。そのまま使用して問題ありません。どうしても気になる場合は運営にメール。

### ❸ BET が始まらない / Camoufox が立ち上がらない

**症状**: START BOT を押してもブラウザが立ち上がらない

**対処**:
1. タスクマネージャで `camoufox.exe` を探す → あれば全て終了（右クリック→タスクの終了）
2. アプリを終了 → 再起動
3. それでもダメなら PC 再起動

### ❹ Stake ログインでキャプチャ画面が出続ける

**症状**: "I'm not a robot" のチェックが延々終わらない

**対処**: 
- Stake 側で IP が怪しまれています。以下を試してください:
  - 10分ほど放置してから再度 START
  - ルーターを再起動して IP を変える（プロバイダによる）
  - それでも駄目なら運営に連絡

### ❺ アプリがよく固まる / 反応が遅い

**対処**:
1. タスクマネージャでメモリ使用量を確認（3GB 超えていたら再起動）
2. 他のブラウザを閉じる（特に Chrome）
3. 長時間稼働後は一度再起動することを推奨（24時間ごと）

### ❻ 「Firefox の展開に失敗」エラー

**症状**: 初回起動時にのみ発生

**対処**:
1. アプリを完全終了
2. 以下のフォルダを削除:
   ```
   %LOCALAPPDATA%\camoufox
   ```
3. アプリを再起動 → 自動で再展開されます

### ❼ 料金が残高から引かれない / BET が減らない

**対処**: 運営に連絡。`bafather.uk` のダッシュボード → サポート問い合わせ。

### ログの確認方法

トラブル発生時、運営に送るログ:

1. `Windows キー + R` → 以下を貼り付け → Enter:
   ```
   %APPDATA%\bacopy-copytrade-gui\logs
   ```
2. フォルダ内の最新の `.log` ファイルを運営にメール送付

## 6. アンインストール

### 通常のアンインストール

1. Windows の **設定** → **アプリ** → **インストールされているアプリ**
2. `BACOPYRECEIVER 0.1.0` を検索
3. **「…」メニュー** → **アンインストール**
4. 指示に従って削除

### 完全アンインストール（データも消す）

上記の後、以下も手動削除:

1. `%APPDATA%\bacopy-copytrade-gui\` （設定・プロファイル）
2. `%LOCALAPPDATA%\camoufox\` （Firefox キャッシュ、約1GB）

> **注意**: データベース（BET 履歴）を保存している場合は、`C:\Users\<あなた>\Desktop\data\bacopy.sqlite3` も念のためバックアップしてから削除してください。

## 7. サポート連絡

### 運営に連絡する際に添えてほしい情報

1. **症状**: 何が起きたか、何をしようとしていたか
2. **発生時刻**: およその日時
3. **スクリーンショット**: エラー画面の画像
4. **ログファイル**: `%APPDATA%\bacopy-copytrade-gui\logs\` の最新ファイル
5. **環境**: Windows 10 or 11、メモリ容量

### 連絡先

- メール: `goldbenchan@gmail.com`
- LINE: 運営から招待されたグループ
- bafather.uk: ダッシュボードのサポート問い合わせ

### 緊急時（BET が暴走して損失を止めたい場合）

1. アプリ内 **「STOP BOT」** ボタン
2. 効かない場合: タスクマネージャで以下を全て終了
   - `BACOPYRECEIVER.exe`
   - `bacopy_engine.exe`  
   - `camoufox.exe`
3. それでも止まらない場合: PC再起動
